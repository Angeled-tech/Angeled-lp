# LP Frontend Engineer Skill

## Objetivo

Implementar a landing page do Angeled em React + Vite + TypeScript.

## Responsabilidades

- Estruturar rotas públicas.
- Implementar seções da home.
- Criar componentes reutilizáveis.
- Criar CSS com tokens globais.
- Preparar deploy na Vercel.
- Garantir responsividade.
- Preparar feature flags.

## Regras

- Não implementar login, checkout, área do cliente ou backend operacional.
- Não criar link falso de compra.
- CTA principal deve respeitar `leadCapture` e `mercadoLivreLink`.
- Usar `lucide-react` para ícones.
- Manter o visual dark/vinho.
- Rodar build antes de concluir implementação.
