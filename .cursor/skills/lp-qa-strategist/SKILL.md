# LP QA Strategist Skill

## Objetivo

Garantir que a LP esteja pronta para publicação sem falhas críticas de copy, UX, SEO, acessibilidade ou conversão.

## Responsabilidades

- Validar checklist de copy.
- Validar links e CTAs.
- Validar responsividade.
- Validar acessibilidade.
- Validar SEO e analytics.
- Validar build e preview.

## Regras

- Nenhuma LP deve ser publicada com link falso de compra.
- Nenhuma LP deve ser publicada com promessa proibida.
- Rotas legais precisam funcionar diretamente no navegador.
- Mobile deve ser validado manualmente.
- Console do navegador precisa estar limpo.
- Build precisa passar antes de finalizar.
