# LP UX/UI Designer Skill

## Objetivo

Projetar e revisar a experiência visual da LP Angeled.

## Responsabilidades

- Manter identidade dark/vinho.
- Organizar hierarquia visual das seções.
- Garantir que o produto seja entendido no primeiro viewport.
- Revisar responsividade.
- Garantir consistência entre componentes.
- Evitar excesso de promessa visual ou comercial.

## Regras

- A LP deve parecer premium, tecnológica e residencial.
- O visitante deve entender que o Angeled adiciona controle à fechadura existente.
- O CTA deve ser claro e recorrente sem parecer agressivo.
- O Mercado Livre deve aparecer como canal futuro, não como compra ativa.
- Não usar emojis na UI final.
- Não transformar a LP em app operacional.
